Spring Boot AJAX Board
스프링 부트 비동기 게시판
화면 깜빡임 없이 부드럽게 동작하는 100% AJAX 기반의 게시판 및 회원 관리 웹 서비스입니다.

사용언어
Java

HTML / CSS

JavaScript

사용 라이브러리
jQuery

MyBatis

사용 프레임워크
Spring Boot (3.5.8)

Spring Security

데이터베이스
MySQL

개발환경
Windows / MAC OS

IntelliJ IDEA / Eclipse

Java 17

핵심 기능
회원 관리 기능

100% AJAX 기반 화면 깜빡임 없는 회원가입 및 로그인

가입 시 아이디 및 이메일 실시간 중복 검사

계정 찾기 및 임시 비밀번호 이메일 발송 기능 (Spring Mail)

BCrypt를 활용한 안전한 비밀번호 암호화 보관

게시판 기능

비동기 통신(AJAX)을 이용한 게시글 등록, 삭제, 수정, 확인 (CRUD)

동적 쿼리를 활용한 게시글 검색 기능 (제목, 내용, 작성자)

게시글 리스트 페이징 처리 기능

회원 및 비회원(Guest) 권한 분리
(비회원은 작성 시 입력한 개별 비밀번호로 수정/삭제 권한 검증)
