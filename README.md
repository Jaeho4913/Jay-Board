# Spring Boot Board

**사이트 주소:** http://110.165.19.54/

Spring Boot 기반의 AJAX 비동기 웹 게시판 프로젝트입니다.
게시글 CRUD, 회원 관리, 좋아요, 댓글, 검색, 페이징, 정렬 기능을 직접 구현하며 웹 게시판의 기본 흐름과 권한 처리 구조를 학습하기 위해 제작했습니다.

---

## 프로젝트 소개

`Spring Boot Board`는 게시글 작성부터 상세 조회, 수정, 삭제, 댓글 작성 및 수정/삭제, 좋아요, 검색, 정렬, 페이징까지 포함한 게시판 웹 애플리케이션입니다.

회원과 비회원의 권한을 분리하여 처리하고, AJAX 기반 비동기 요청을 활용해 화면 전환을 최소화한 게시판 기능을 구현했습니다.

## 주요 구현 포인트

### AJAX 기반 비동기 처리

게시글 목록 조회, 상세 조회, 좋아요, 댓글 작성 / 수정 / 삭제 기능을 AJAX 기반으로 구현했습니다.
페이지 전체를 새로고침하지 않고 필요한 영역만 갱신하도록 구성했습니다.

### MyBatis 기반 SQL 분리

Mapper XML을 활용하여 SQL을 분리하고, 게시글 검색 / 정렬 / 페이징 / 댓글 수 조회 로직을 관리했습니다.

### 서버단 권한 검증

프론트에서 버튼을 숨기는 것과 별개로, 서버에서 실제 작성자와 로그인 사용자를 비교하여 수정 / 삭제 권한을 검증했습니다.

### 댓글 수 집계

게시글 목록 조회 시 댓글 테이블을 기준으로 게시글별 댓글 수를 계산하여 목록 화면에 표시하고, 댓글 수 기준 정렬 기능을 추가했습니다.
